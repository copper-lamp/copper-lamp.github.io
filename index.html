安装指南

安装指南

快速开始使用Copper Lamp投影模组

1

安装前置

确保已安装LeviLamina框架：

# 下载LeviLamina git clone https://github.com/LiteLDev/LeviLamina.git cd LeviLamina xmake f -m release xmake -j8
2

下载模组

从GitHub获取最新版Copper Lamp：

访问GitHub下载
3

安装模组

将模组放入服务器插件目录：

# 创建插件目录 mkdir plugins/CopperLamp # 复制模组文件 cp CopperLamp.dll plugins/CopperLamp/ cp -r resources/ plugins/CopperLamp/
4

配置文件夹

创建必要的资源文件夹：

# 个人投影文件夹 mkdir plugins/CopperLamp/resources/structures # 共享投影文件夹 mkdir plugins/CopperLamp/resources/shared_structures
5

启动服务器

启动Minecraft服务器，模组将自动加载：

# 启动BDS服务器 ./bedrock_server # 检查日志 [INFO] CopperLamp 已成功加载
6

添加投影文件

将结构文件放入对应文件夹：

  • 个人投影：plugins/CopperLamp/resources/structures/
  • 共享投影：plugins/CopperLamp/resources/shared_structures/
使用教程

使用教程

掌握Copper Lamp的核心功能

基本工作流程

1

加载投影

使用命令加载投影文件：

/projection load 红石大门 /projection load 自动农场 true # 加载为共享投影
2

设置投影中心

手持木棍，蹲下并右键点击一个方块：

[系统] 投影中心已设置为: (123, 64, 456)
3

开始建造

投影显示需要放置的方块位置：

  • 正确放置：投影消失
  • 错误放置：显示提示框
4

使用轻松放置

在投影位置放置方块时：

  • 自动选择正确方块
  • 减少背包数量
  • 返还原始方块

高级功能

分层建造

使用方向键切换投影层：

↑ 键：上一层 ↓ 键：下一层 /projection layer 5 # 跳转到第5层

团队协作

多人共享同一投影：

# 保存为共享投影 /sharedprojection save 团队建筑 # 加载共享投影 /projection load 团队建筑 true

材料清单

查看建造所需材料：

/projection materials 红石计算机 石头 x 128 橡木木板 x 64 红石粉 x 32 红石中继器 x 12 粘性活塞 x 8

错误修正

当显示错误提示时：

  • 破坏错误方块
  • 投影会自动恢复显示
  • 重新放置正确方块

常见问题

常见问题解答

解决您在使用Copper Lamp时遇到的问题

Q: 如何创建投影文件？

A: 在创造模式中使用结构方块保存建筑，然后将生成的.mcstructure文件放入plugins/CopperLamp/resources/structures/文件夹中。模组会自动转换格式。

Q: 为什么投影不显示？

A: 请检查：1) 是否使用木棍设置了投影中心；2) 是否在正确的层（使用方向键切换）；3) 是否距离投影中心过远；4) 是否安装了LeviLamina前置。

Q: 轻松放置功能无效怎么办？

A: 首先使用 /easyplace 检查是否启用。如果已启用但无效，请确认：1) 背包中有正确的方块；2) 方块方向正确；3) 投影已加载。

Q: 如何与其他玩家共享我的建筑？

A: 1) 保存投影：/sharedprojection save 建筑名称；2) 其他玩家加载：/projection load 建筑名称 true。所有玩家需要设置相同的投影中心点。

Q: 错误提示是什么意思？

A: 表示该位置放置了错误的方块。破坏该方块后提示会消失，投影会重新显示。重新放置正确的方块即可。

Q: 模组会影响服务器性能吗？

A: 大型投影可能有一定性能影响。建议：1) 只加载需要的投影；2) 远离时自动卸载；3) 使用分层显示；4) 简化复杂结构。

Q: 是否支持跨维度投影？

A: 当前版本仅支持主世界投影。跨维度功能正在开发中，预计v1.3版本支持。

Q: 如何获得更多帮助？

A: 访问我们的GitHub仓库：https://github.com/copper-lamp/copper-lamp 提交issue或加入Discord社区获取支持。
