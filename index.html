CopperLamp 介绍

CopperLamp 是一个服务于 Minecraft 基岩版专用服务器(BDS)的插件模组，可以让你更清晰地了解到游戏是如何运行的。作为专为 MCBE 生电服务的 BDS 模组，它提供了强大的调试和优化功能。

CopperLamp 是 traptoor 的精神继承者，加入了空手开盒等实用功能，并提供了 GUI 使其更加便于使用，搭载的功能都是精简实用的。

支持版本

当前版本 1.1 支持 Minecraft 基岩版 1.21.9 及以上版本。

# 前置组件
LeviLamina: https://github.com/LiteLDev/LeviLamina
legacy-script-engine-quickjs: https://github.com/LiteLDev/LegacyScriptEngine

核心功能

游戏性能监控

实时查看 MSPT 和 TPS，分析游戏运行性能，监控实体数量，确保服务器流畅运行。

村庄机制可视化

显示村庄边界、床位、工作站点等关键信息，帮助优化村庄设计和铁傀儡农场。

假人系统

创建和管理假人，模拟玩家行为，用于测试农场效率和加载区块。

生物生成分析

显示硬编码结构刷怪区域(HSA)，查看史莱姆区块，优化刷怪塔设计。

游戏速度控制

调整游戏运行速率，支持步进执行和暂停，便于精确调试和测试。

投影功能

提供强大的投影功能，逐层显示建筑结构，便于大型建筑复制和设计。

安装指南

1

安装前置组件

确保你的服务器已安装以下前置组件：

# LeviLamina
https://github.com/LiteLDev/LeviLamina

# legacy-script-engine-quickjs
https://github.com/LiteLDev/LegacyScriptEngine
2

下载 CopperLamp

访问 GitHub 仓库下载最新版本的插件：

https://github.com/copper-lamp/CopperLamp

选择与你的服务器版本兼容的插件版本。

3

安装插件

将下载的插件文件放入服务器的插件目录：

# 路径示例
/plugins/CopperLamp/
4

启动服务器

启动你的 Minecraft 基岩版服务器，插件将自动加载。

在控制台输入 clp gui 或在游戏中输入 /clp gui 来打开假人控制面板，验证插件是否安装成功。

CopperLamp 提供了一系列强大的指令来访问其功能：

常见问题

插件安装后没有生效怎么办？

1. 确认已安装所有前置组件（LeviLamina 和 legacy-script-engine-quickjs）

2. 检查插件文件是否放置在正确的目录中

3. 查看服务器日志是否有加载错误

4. 确保服务器版本与插件版本兼容

自由视角模式如何退出？

在自由视角模式下，按下跳跃键（默认空格）可以上升，潜行键（默认Shift）可以下降。要退出自由视角，只需再次输入 /fcc 命令即可。

假人系统需要什么权限？

默认情况下，只有管理员可以使用假人系统。你可以通过 /clp op false 命令允许普通玩家使用假人功能。

如何调整游戏速度？

使用 /tick rate 命令调整游戏速度：

- /tick rate 1：正常速度

- /tick rate 2：2倍速度

- /tick rate 0.5：半速

- /tick free：恢复正常速度
